Position Sizing Algorithm

Position Size = (Account Balance × Risk %) / (Entry Price - Stop Loss) Confidence Factor = Signal Confidence / 100 Adjusted Size = Position Size × Confidence Factor Final Size = min(Adjusted Size, Max Position Limit)

📖 Usage Instructions

🚀 Getting Started

Quick Start: Follow these steps to run the Gold Trading AI application

Prerequisites

  • Python 3.8 or higher
  • Windows 10/11, macOS, or Linux
  • Minimum 4GB RAM, 8GB recommended
  • Internet connection for data feeds

Installation Steps

# Clone or download the project cd "Gold Trading AI Project" # Install dependencies pip install -r requirements.txt # Run the application python main.py

Launch Options

# Standard launch python main.py # Alternative GUI python gui_app/main_application.py # Demo mode python demo.py # Comprehensive launcher python launch.py --quick

Interface Overview

  • 🥇 Gold AI Trading Tab: Main analysis and signal generation
  • 📊 Technical Analysis Tab: Detailed technical indicators
  • 📈 Fundamental Analysis Tab: Economic data and sentiment
  • ⚖️ Risk Management Tab: Position sizing and risk metrics
  • 💾 Database & History Tab: Historical data and performance

Step-by-Step Process

  1. Launch Application: Open the Gold Trading AI interface
  2. Click "🚀 ANALYZE GOLD MARKET": Start the analysis process
  3. Wait for Analysis: System will process data (2-6 seconds)
  4. Review Results: Check signal, confidence, and risk metrics
  5. Make Trading Decision: Use signals for manual execution

Signal Interpretation

Signal Type Meaning Action Risk Level
STRONG_BUY High confidence bullish signal Consider long position Medium
BUY Moderate bullish signal Consider small long position Low-Medium
HOLD Neutral or uncertain market Wait for clearer signals Low
SELL Moderate bearish signal Consider small short position Low-Medium
STRONG_SELL High confidence bearish signal Consider short position Medium

Enabling Real-Time Analysis

  1. Check the "⚡ REAL-TIME MODE" checkbox
  2. System will analyze market every 60 seconds
  3. Signals will update automatically
  4. Monitor for signal changes and confirmations
⚠️ Note: Real-time mode increases API usage and may hit rate limits with free data sources.

📊 Configuration Options

# Configuration file: config/settings.yaml # Risk Management Settings risk_management: max_risk_per_trade: 0.02 # 2% max_portfolio_risk: 0.10 # 10% max_drawdown: 0.05 # 5% min_risk_reward_ratio: 2.0 # 2:1 # Data Sources data_sources: primary: "yfinance" symbols: ["GC=F", "XAUUSD=X", "GLD"] real_time_interval: 30 # seconds # Model Settings ml_models: target_accuracy: 0.90 # 90% confidence_threshold: 0.60 # 60% ensemble_weights: "auto"
